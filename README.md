# RapidReel Creations Portfolio

This is the official portfolio website for RapidReel Creations, built with Next.js, Tailwind CSS, and Framer Motion.

## Features
- **Cinematic Parallax Gallery**: High-performance scrolling animations.
- **Dark Mode Aesthetic**: Premium, video-centric design.
- **SEO Optimized**: JSON-LD Structured Data, Semantic HTML, and optimized Metadata.
- **Responsive**: Mobile-first design architecture.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS, Aceternity UI, Shadcn/UI
- **Animation**: Framer Motion
- **Language**: TypeScript
